More Clipboard for Eclipse update site

  • To use the update site paste the link to the Eclipse Update Manager: http://moreclipboard.sourceforge.net/updates
  • To manually download the plugin files go to https://sourceforge.net/projects/moreclipboard/files/

Note: If you have previously installed More Clipboard v1.0.1 you will have to manually delete the ECLIPSE_ROOT/plugins/MoreClipboard_1.0.1.jar
